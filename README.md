# 🔐 Modern Login Page

A clean and modern **Login & Signup UI** built using HTML, CSS, and JavaScript.  
This project features a smooth and responsive authentication interface with an attractive design.

---

## 🚀 Features

- 🔑 Login & Signup form  
- 🔄 Smooth toggle animation between forms  
- 👁️ Show/Hide password option  
- 📱 Fully responsive design  
- 🎨 Modern UI with gradients & effects  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript
